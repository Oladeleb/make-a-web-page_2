HTML

Hyper Text Markup Language (HTML) is use to write webpages. HTML is made up of "text content", what you see, and "mark up" which describes how "what you see" looks like, references to links, images, video, etc. This video gives you a good overview.

Tags and Elements

HTML tags are used to mark the beginning and/or closing of an element. The name of the opening and closing tag must be an exact match. Opening tag determines the HTML output. HTML elements comprises of opening and closing tags with content.

Inline Vs... Block Elements

Inline elements are displayed without a line break, so you can have multiple inline elements on the same horizontal line of a page. Block elements are displayed with a line break, so you can't have multiple block elements on the same horizontal line unless you declare display: flex. All elements, inline and block, have an invisible box around them that you can target via CSS.

Stage 2 : Creating Structure Document with HTML

Developer Tools ( in the Browser)

HTML page structure comprises of three parts: HTML version, a Head section and the Body. HTML version declares what version of HTML is being used. The Head section contains metadata about the current document, such as, cascading style sheet (css), document title, etc. The Body section contains the page content that is visible to users.

To view a website page structure, use google chrome developer tools, for example, to show the website page structure as the browser interpret it. As you mouse over an element within a structure, the section of the page is highlighted for readiability.

The "Tree-like Structure of HTML"

Every browser builds a tree-like structure of a page. HTML tree is hierarchical in nature, where some of the root(parent) attributes are inherited by it dependents(child) and this can be seen by expanding or collapsing the parent record to show or hide related content. The DOM(Document Object Model) defines the tree-structure of a page.
